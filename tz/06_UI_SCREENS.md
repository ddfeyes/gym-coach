# UI/UX Специфікація — Telegram Mini App

## Загальні правила дизайну

### Тема
- Dark theme (як в outreach-agent)
- Фон: #0f0f0f (основний), #1a1a1a (картки), #252525 (hover)
- Текст: #ffffff (основний), #a0a0a0 (secondary), #666666 (muted)
- Акцент: #6366f1 (indigo — кнопки, прогрес-бари)
- Успіх: #22c55e (зелений — PR, виконано)
- Попередження: #f59e0b (жовтий — увага, адаптація)
- Небезпека: #ef4444 (червоний — біль, стоп)
- Бордери: #2a2a2a
- Шрифт: system font (Telegram дефолт)

### Компоненти
- Картки з rounded corners (12px)
- Subtle shadows (rgba(0,0,0,0.3))
- Smooth transitions (200ms)
- Haptic feedback через Telegram WebApp API
- Bottom navigation bar з іконками

### S-000: Bottom Navigation

```

│  🏋️ Training  │  💬 Chat  │  📊 Progress  │  ⚙️ More  │
```

## Telegram WebApp API Usage

```javascript
// Ініціалізація
const tg = window.Telegram.WebApp;
tg.ready();
tg.expand(); // full height

// Тема — підхоплюємо з Telegram якщо можливо
const theme = tg.colorScheme; // 'dark' або 'light'

// Haptic feedback
tg.HapticFeedback.impactOccurred('light');   // логування підходу
tg.HapticFeedback.impactOccurred('medium');  // завершення вправи
tg.HapticFeedback.notificationOccurred('success'); // PR
tg.HapticFeedback.notificationOccurred('warning'); // AI попередження

// Main button (опціонально для key actions)
tg.MainButton.setText('Почати тренування');
tg.MainButton.show();
tg.MainButton.onClick(() => startWorkout());

// Back button
tg.BackButton.show();
tg.BackButton.onClick(() => goBack());

// User data for auth
const initData = tg.initData; // для валідації на бекенді
```
